# 唤星 HuanXing 品牌官网

## 项目概述

这是唤星（HuanXing）的品牌官网，一个 AI 数字分身产品的营销落地页。

## 设计参考

请仔细阅读以下文档，严格遵循品牌规范：

- `HUANXING-BRAND.md` — 产品定义与品牌设计（定位、用户、商业模式）
- `HUANXING-VISION.md` — 品牌愿景（使命、价值观、品牌故事）
- `HUANXING-UI-SPEC.md` — UI 设计规范（色彩、字体、间距、组件、动效）

## 技术栈

- React 18 + TypeScript
- Vite 构建
- Tailwind CSS 4
- Framer Motion 动效
- 纯前端静态站点，无需后端

## 品牌色彩（必须严格遵循）

```
主色：#6C5CE7（星紫）
辅色：#00D2FF（星蓝）
强调色：#FFD93D（星光金）
```

## 页面结构

### 1. Hero 区域
- 全屏深空黑背景 + 星光粒子动画
- 大标题："唤醒你的星，AI 与你共生"
- 副标题：简短产品描述
- CTA 按钮："开始唤星"（品牌渐变按钮）
- 中心有一颗正在发光的星的动画效果

### 2. 产品特性区
- 三大核心价值卡片：
  1. 养成感 — 你的分身由你塑造，越用越像你
  2. 全时在线 — 7×24 后台运行，永不离线
  3. 无限可能 — 连接工具、管理信息、执行任务
- 每个卡片有图标 + 标题 + 描述
- 卡片背景 #161B22，hover 时微弱星光发光

### 3. 养成系统展示
- 展示四个进阶等级：星尘 → 星芒 → 星辰 → 星耀
- 每个等级有对应的星光亮度和颜色
- 星尘：#6E7681（暗淡）
- 星芒：#6C5CE7（星紫）
- 星辰：#00D2FF（星蓝）
- 星耀：#FFD93D（星光金）
- 用动画展示从暗到亮的进化过程

### 4. 产品界面预览
- 模拟聊天界面截图/动画
- 展示分身与用户的对话场景
- 深色主题，符合 UI 规范中的聊天气泡样式

### 5. 定价区域
- 四档订阅卡片：
  - 星尘（免费）：每日20次对话，7天记忆
  - 星芒（¥29/月）：每日200次对话，长期记忆 — 标记为"推荐"
  - 星辰（¥79/月）：无限对话，完整记忆，工具连接
  - 星耀（¥199/月）：一切无限，专属模型，API接口
- 推荐档位用品牌渐变边框高亮

### 6. 品牌故事区
- "每个人小时候都仰望过星空..."
- 简短的品牌故事叙述
- 配合星空背景动画

### 7. Footer
- 品牌 Logo + 名称
- 导航链接
- 社交媒体图标
- 版权信息

## 动效要求

- 星光粒子：微小光点随机漂浮，颜色 #FFD93D，opacity 0.3~0.8
- 滚动动画：各区域滚动进入时有淡入 + 上移效果
- 星光呼吸：关键元素有 opacity 0.6~1.0 的呼吸动画
- 按钮 hover：微弱发光效果
- 整体流畅自然，不要过度动画
