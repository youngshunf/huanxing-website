# 唤星 HuanXing — 产品定义与品牌设计

---

## 一、产品定义

### 1.1 一句话定义

> **唤星 — 人人都能拥有的 AI 数字分身**

### 1.2 核心理念

每个人都值得拥有一个懂自己的 AI 分身。它不是冷冰冰的工具，而是一个由你唤醒、由你养成、与你共同成长的数字存在。

### 1.3 产品定位

| 维度 | 定义 |
|------|------|
| 品类 | AI 数字分身平台 |
| 目标 | 人人都能用，人人都需要 |
| 核心体验 | 养成式 AI 分身，7×24 在线 |
| 差异化 | 不是千人一面的助手，是独一无二的"你的分身" |

### 1.4 目标用户

**主力用户：18-45 岁，对 AI 好奇但被技术门槛挡住的人**

- 上班族：需要工作助手、日程管理、信息整理
- 学生：学习辅助、知识管理、创作灵感
- 自由职业者：效率工具、客户沟通、内容创作
- 数码爱好者：喜欢尝鲜、愿意"养成"自己的 AI
- 普通用户：想要一个随时能聊、能帮忙的数字伙伴

**共同特征**：不想折腾技术配置，希望开箱即用，但又希望 AI 能越来越懂自己。

### 1.5 核心价值主张

```
传统 AI 助手：你问它答，千人一面，用完即走
唤星数字分身：与你共生，越用越懂你，7×24 不下线
```

**三大核心价值**：

1. **养成感** — 你的分身由你塑造，记住你的偏好、习惯、风格，越用越像"你"
2. **全时在线** — 7×24 小时后台运行，不错过任何重要信息，主动为你工作
3. **无限可能** — 不限于聊天，能连接你的工具、管理你的信息、执行你的任务

## 二、品牌设计

### 2.1 品牌名称

**中文名**：唤星
**英文名**：HuanXing
**含义**：唤醒属于你的那颗星。每个人心中都有一颗星，唤星帮你点亮它。

**命名逻辑**：
- "唤" — 唤醒、召唤，暗示用户主动激活自己的 AI 分身
- "星" — 星辰、星光，代表无限可能和未来感
- 组合 — 唤醒星辰，寓意每个人都能拥有属于自己的智能星体

### 2.2 品牌调性

**关键词**：科技感 · 未来感 · 个人化 · 成长性

| 调性维度 | 方向 | 说明 |
|----------|------|------|
| 视觉 | 深空 + 星光 | 深色底色 + 星辰光效，科技而不冰冷 |
| 语言 | 简洁有力 | 不说废话，但有温度 |
| 交互 | 流畅灵动 | 像星光流动，响应迅速 |
| 情感 | 陪伴成长 | 不是工具感，是"我的分身在进化" |

### 2.3 品牌色彩

**主色调**：星空渐变

```
主色：#6C5CE7（星紫）— 科技感、神秘感
辅色：#00D2FF（星蓝）— 未来感、清澈
强调色：#FFD93D（星光金）— 点亮、唤醒
背景色：#0D1117（深空黑）— 沉浸、专注
文字色：#E6EDF3（星白）— 清晰、干净
```

**渐变方案**：
- 主渐变：#6C5CE7 → #00D2FF（星紫到星蓝，用于品牌标识和关键元素）
- 背景渐变：#0D1117 → #161B22（深空层次感）
- 光效：#FFD93D 的微光粒子效果，模拟星光

### 2.4 品牌标识概念

**Logo 概念**：一颗正在被"唤醒"的星

```
设计方向：
- 简约的星形轮廓
- 中心有一个光点正在扩散（唤醒的瞬间）
- 可以抽象为：一个圆（星体）+ 向外扩散的光芒
- 整体风格：几何、简洁、有动感
```

**图标变体**：
- 完整版：星形 + "唤星" 文字
- 简约版：仅星形图标（用于 App 图标、系统托盘）
- 动态版：星光闪烁动画（用于启动画面、加载状态）

### 2.5 品牌口号

**主口号**：
> **唤醒你的星，AI 与你共生**

**备选口号**：
- "每个人，都值得一颗星" — 强调普惠
- "你的分身，你的星" — 强调归属感
- "唤星而来，为你而生" — 强调个人化
- "养一颗星，它会发光" — 强调养成感

### 2.6 品牌声音（Brand Voice）

| 场景 | 风格 | 示例 |
|------|------|------|
| 产品介绍 | 简洁有力，有未来感 | "唤星，你的 AI 数字分身。" |
| 功能说明 | 直白易懂，不卖弄 | "它记住你说过的每句话，下次不用重复。" |
| 引导文案 | 温暖鼓励，有陪伴感 | "你的星正在苏醒，给它起个名字吧。" |
| 错误提示 | 轻松不慌，给方案 | "信号不太好，你的星正在努力连接中..." |
| 营销推广 | 有想象力，引发共鸣 | "如果你能复制一个自己，你会让它做什么？" |

## 三、产品体验设计

### 3.1 首次体验流程（FTUE）

```
下载安装
  ↓
启动 — 星空动画，一颗暗淡的星缓缓出现
  ↓
"唤醒你的星" — 手机号登录
  ↓
"给你的星起个名字" — 为 AI 分身命名
  ↓
"告诉它你是谁" — 简单的个人偏好设置
  ├─ 你希望它叫你什么？
  ├─ 你的工作/身份？（可跳过）
  └─ 你最希望它帮你做什么？（多选）
  ↓
"你的星已苏醒" — 星光亮起动画
  ↓
进入聊天 — 分身发出第一条消息
  "你好，我是 [名字]，你的星刚刚亮起。以后我会一直在这里。"
```

### 3.3 养成系统

**分身成长维度**：

| 维度 | 说明 | 体现 |
|------|------|------|
| 记忆深度 | 记住的事情越来越多 | "你上次说喜欢吃川菜" |
| 风格适配 | 说话方式越来越像你 | 语气、用词、幽默感 |
| 主动性 | 从被动回答到主动提醒 | "明天有雨，要不要我帮你改行程？" |
| 能力扩展 | 连接更多工具和服务 | 邮件、日历、笔记、智能家居 |

**成长可视化**：
- 星光亮度随使用时间增加（暗淡 → 明亮 → 闪耀）
- 分身等级：星尘 → 星芒 → 星辰 → 星耀
- 里程碑提示："你的星已经记住了 100 件关于你的事"

### 3.4 系统托盘 & 后台运行

```
系统托盘图标：一颗小星星
  ├─ 亮起 = 在线工作中
  ├─ 微闪 = 有新消息/提醒
  └─ 暗淡 = 待机中

右键菜单：
  ├─ 打开唤星
  ├─ 今日摘要
  ├─ ──────────
  ├─ 状态：在线
  ├─ ──────────
  ├─ 设置
  └─ 退出
```

## 四、商业模式

### 4.1 订阅体系

| 层级 | 价格 | 内容 |
|------|------|------|
| 免费版（星尘） | ¥0 | 每日 20 次对话，基础记忆（7天），单设备 |
| 标准版（星芒） | ¥29/月 | 每日 200 次对话，长期记忆，多设备同步 |
| 专业版（星辰） | ¥79/月 | 无限对话，完整记忆，工具连接，优先响应 |
| 旗舰版（星耀） | ¥199/月 | 一切无限，专属模型，API 接口，多分身 |

### 4.2 定价逻辑

- 免费版足够体验核心价值，形成养成习惯
- ¥29 的标准版是主力转化目标，门槛低
- 记忆是核心付费驱动力 — 免费版只保留 7 天记忆，付费版永久
- "你的星会忘记你"是最强的付费动机

### 4.3 增长策略

**冷启动**：
1. 科技社区种子用户（少数派、V2EX、即刻）
2. 短视频展示"AI 分身"的惊艳时刻
3. 邀请制 + 内测资格，制造稀缺感

**增长飞轮**：
```
用户养成分身 → 分身越来越懂用户 → 用户离不开 → 付费留住记忆 → 推荐给朋友
```

## 五、竞品差异化

| 维度 | 豆包/Kimi/通义 | 唤星 |
|------|----------------|------|
| 定位 | AI 对话工具 | AI 数字分身 |
| 记忆 | 有限/无 | 核心特性，持续成长 |
| 个性化 | 千人一面 | 由用户养成塑造 |
| 运行模式 | 打开才能用 | 7×24 后台运行 |
| 主动性 | 被动回答 | 主动提醒和工作 |
| 情感连接 | 弱 | 强（养成感+成长可视化） |
| 工具连接 | 平台内 | 开放（本地+云端） |

**一句话差异**：别人做的是"更聪明的搜索框"，唤星做的是"另一个你"。
